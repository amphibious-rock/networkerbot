Thank you for using Networker Bot. This page includes frequently asked questions and common errors experienced when using the bot.

/gwr_info

What you input here must be a number, and it refers to Pre Grouping GWR, not post privatisation GWR. an example of a valid input would be: "5700". "801" would not be accepted as that is not a GWR class of locomotive, that falls under the TOPS numbering scheme as it is refers to post privatisation GWR.

/leaderboard-user

This will only accept a username input, it should work when you are in a server with the target user (including yourself) in, or group chat or DM

/train_image & /train_info

This must be a integer of a TOPS class. This is only the class and not the full number, "465" would be accepted, whilst "465908" would not.

If you have any other questions , please contact us in the official discord server .
